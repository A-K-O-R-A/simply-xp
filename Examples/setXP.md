# setXP

Set XP to a user | `setXP`

### Usage

```js
let xp = require('simply-xp')

xp.setXP(userID, guildID, xp)
```

### Example

```js
let xp = require('simply-xp')

xp.setXP(message.author.id, message.guild.id, 100)
```

- ## Returns `<Object>`

```
{
  xp: 100
}
```
